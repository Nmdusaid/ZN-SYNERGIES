# **App Name**: ZN Synergies

## Core Features:

- Predictive Logistics Tracker: An AI-powered tool that calculates intelligent ETAs and optimizes routing paths based on live environmental variables and port status.
- Interactive Global Service Grid: Premium animated glassmorphism cards showcasing Air, Sea, and Road logistics with advanced hover glow and lift effects.
- Real-time Route Visualization: A dynamic 3D world map showing live shipment movements and network connectivity across global territories.
- Smart Freight Calculator: A tool that uses generative logic to provide dynamic price estimates for complex supply chain solutions and global enterprise movements.
- Fleet Parallax Showcase: Immersive scroll-based gallery displaying ZN Synergies' modern trucks, cargo ships, and aircraft with cinematic motion effects.
- Advanced Logistics API Connector: Enterprise-grade database integration to synchronize live cargo status and customs clearance history.
- High-Impact Performance HUD: Live-updating statistics dashboard featuring animated counter effects for deliveries, countries reached, and on-time percentage.

## Style Guidelines:

- The primary color is a polished Tech Silver (#A1ABB8), reflecting industrial precision and high-end tech. The background is a Deep Charcoal (#0B0C0E), achieving a dark-scheme aesthetic with subtle saturation. The accent is a Soft Violet (#B8BCF2), adding a modern futuristic touch that contrasts sharply with the monochromatic base.
- The 'Inter' typeface is used throughout for its modern, machined, and neutral aesthetic. It is suitable for both high-impact headlines and readable data strings in the logistics dashboard. Note: currently only Google Fonts are supported.
- Custom Lucide React icons rendered in a silver stroke with a subtle drop-glow to match the futuristic logistics theme.
- An expansive, minimalist grid with large whitespace and cinematic 'Framer Motion' transitions to create an elite, breathable user experience.
- Utilizes GSAP and Lenis for ultra-smooth scrolling, scroll-triggered text reveals, and high-performance glassmorphism hover animations.
